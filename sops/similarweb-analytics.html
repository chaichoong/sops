🎯 1. Purpose & Scope

Retrieves and presents comprehensive website analytics for any domain using the SimilarWeb API. Covers total visits, unique visitors, bounce rate, global ranking, traffic sources (desktop and mobile), and geographic distribution. Saves all data to files immediately to protect against API credit depletion mid-run.

ℹ️
Scope: In scope: any publicly trackable domain, up to 12 months of historical data, monthly granularity. Out of scope: real-time data, daily granularity, subdomains (if main_domain_only=True), or private/unlisted domains.

💡 2. When to Use This Skill

💬 Trigger Phrases

Use this skill when Kevin or the team says any of the following:

  • "Analyse [domain]" / "Traffic for [website]"
  • "How popular is [website]?"
  • "Compare [domain A] vs [domain B]"
  • "What are the traffic sources for [domain]?"
  • "Global ranking for [website]"
⚠️
Do not use this skill when: Internal traffic analytics for Kevin's own websites — use Google Analytics or a connected tool for first-party data.

Call SimilarWeb APIs

automated
  1. Import ApiClient from /opt/.manus/.sandbox-runtime.
  2. Call relevant APIs: get_global_rank, get_visits_total, get_unique_visit, get_bounce_rate, get_traffic_sources_desktop, get_traffic_sources_mobile, get_total_traffic_by_country.
  3. Default time ranges: Global Rank / Visits / Unique Visit / Bounce Rate → 6 months. Traffic Sources / By Country → 3 months (max 3 months for by-country).
  4. CRITICAL: Save all retrieved data to files immediately after each API call to prevent data loss from credit depletion.

🚨 8. Error Handling

  • API call fails mid-run (credit depletion): All previously retrieved data is already saved to files. Resume from the next uncalled API.
  • Domain not tracked: SimilarWeb may not have data for low-traffic or new domains. Report this clearly.

🛠️ 9. Troubleshooting

❌ Data missing for a specific month
✅ SimilarWeb only provides data up to the last complete month. The current month is never available.
❌ By-country API returns no results with 6-month range
✅ get_total_traffic_by_country has a 3-month maximum date range. Adjust start_date accordingly.
